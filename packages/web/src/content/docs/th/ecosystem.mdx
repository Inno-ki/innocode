ชุดโครงการชุมชนที่สร้างขึ้นบน InnoCode

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ InnoCode ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-innocode](https://github.com/awesome-innocode/awesome-innocode) และ [innocode.cafe](https://innocode.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
