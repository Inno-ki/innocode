Une collection de projets communautaires construits sur InnoCode.

:::note
Vous souhaitez ajouter votre projet lié à InnoCode à cette liste ? Soumettez un PR.
:::

Vous pouvez également consulter [awesome-innocode](https://github.com/awesome-innocode/awesome-innocode) et [innocode.cafe](https://innocode.cafe), une communauté qui regroupe l'écosystème InnoCode.
